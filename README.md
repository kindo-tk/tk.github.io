# Portfolio - [Tufan Kundu](https://www.linkedin.com/in/tufan-kundu-577945221/)

## Overview
This repository contains the source code for the personal portfolio website of Tufan Kundu, a Data Scientist and Machine Learning Practitioner. The website showcases professional experience, certifications, technical skills, and a collection of data science and machine learning projects.

**Live Site:** [https://kindo-tk.github.io/tk.github.io/](https://kindo-tk.github.io/tk.github.io/)

## Folder Structure
The project follows a standard static website structure:

*   **index.html**: The main entry point of the application, containing the HTML markup and content.
*   **assets/**: A directory containing all static resources:
    *   **css/**: Stylesheets for the application, including the main theme and third-party libraries.
    *   **js/**: JavaScript files for interactivity and third-party plugins.
    *   **img/**: Images used throughout the site, organized by category (hero, project, icons, etc.).
    *   **fonts/**: Custom web fonts.

## Technology Stack
The portfolio is built using the following technologies:

### Core Technologies
*   **HTML5**: For semantic structure and content.
*   **CSS3**: For styling and responsive design.
*   **JavaScript**: For dynamic behavior and interactivity.

### Libraries and Frameworks
*   **Bootstrap 5**: A front-end framework for developing responsive, mobile-first websites.
*   **jQuery**: A JavaScript library for DOM manipulation and event handling.
*   **Isotope**: A library for filtering and sorting layout items (used in Project and Skills sections).
*   **WOW.js**: A library for revealing animations on scroll.
*   **GSAP (GreenSock Animation Platform)**: A robust library for high-performance animations.
*   **Odometer**: A library for smooth number transitions.
*   **Magnific Popup**: A responsive lightbox script.
